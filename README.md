# CoPaw Local Editable Repository

This repository is reconstructed from the installed `copaw==0.0.1` package for local editable development.

## New Frontend

The rebuilt frontend lives in `frontend/` (`React + Vite + TypeScript`).

- Dev: `cd frontend && npm install && npm run dev`
- Build: `cd frontend && npm run build`
- Gray switch: set `COPAW_CONSOLE_STATIC_DIR` to `frontend/dist`
